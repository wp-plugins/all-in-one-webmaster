Add Google, Bing, Yahoo Webmaster Meta Tag to your blog and also add Google Analytics. Automatic sitemap submission in next releases.

== Description ==
If you are a webmaster you probably heard of Google Webmaster Tool. Both Yahoo and Bing have their own versions called "Yahoo SiteExplorer" and "Bing Webmaster Central". All in One Webmaster allow you to add Google, Bing, Yahoo Webmaster Meta Tag into Header of your theme and also you can add Google Analytics into theme footer. It uses the wp_head() and wp_footer() hooks built in to WordPress to add content to your header and/or footer. This is just an initial version. There are so many things which will come out in next releases. i.e. single click automatic sitemap submission, nice GUI, monitor META tags, etc. Please stay tuned for this.

**Features**

* Ability to add Google, Bing, Yahoo Webmaster Meta-Tag
* Ability to add Google Analytics code

== Installation ==
1. Unpack the `download-package`.
2. Upload the file to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Configure the options under Admin Panel `Settings -> All in one Webmaster`.
5. Done and Ready.
6. Configure meta tags and etc in admin page.
